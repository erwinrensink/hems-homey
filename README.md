# Homey HEMS

A Home Energy Management System for Homey Pro.

## Features

- Hoymiles DTU (Modbus TCP)
- Local communication only
- Homey Energy integration
- Advanced Flow support
- Solar curtailment
- Dynamic Energy Prices
- Battery ready
- Heatpump ready
